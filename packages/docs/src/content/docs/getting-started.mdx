## Installation

### Option A: App Store

- **Chrome**: [Chrome Web Store](https://chromewebstore.google.com/detail/context-bro/fgiegklhgcecdiephfipnjdhaojlhgff) — click **Add to Chrome**
- **Firefox**: [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/context-bro/) — click **Add to Firefox**

### Option B: Direct Download (Latest)

Download the latest build directly — always up-to-date, no store review delay:

- [Chrome zip](https://assets.contextbro.app/releases/context-bro-latest-chrome.zip) — unzip, then load in `chrome://extensions` with **Developer mode** → **Load unpacked**
- [Firefox zip](https://assets.contextbro.app/releases/context-bro-latest-firefox.zip) — unzip, then load in `about:debugging` → **Load Temporary Add-on**

### Option C: Build from Source

```bash
git clone https://github.com/DoodleBears/ContextBro.git
cd ContextBro
pnpm install
pnpm run build
```

Then load the unpacked extension from `.output/chrome-mv3/` in `chrome://extensions`.

## Configure Your First Endpoint

1. Click the Context Bro icon → **Settings** (gear icon)
2. Go to the **Endpoints** tab
3. Click **+ Add endpoint**
4. Enter a name (e.g. "My API"), the POST URL, and any headers
5. Toggle it **enabled**
6. Click **Save**

For testing, you can use `https://httpbin.org/post` — it echoes back whatever you send.

## Share a Page

With an endpoint configured, you have several ways to share:

| Method | How |
|--------|-----|
| **Popup** | Click the extension icon → **Share** |
| **Context menu** | Right-click → **Share page to Context Bro** |
| **Keyboard shortcut** | `Ctrl+Shift+K` (or `Cmd+Shift+K` on Mac) |
| **Selection button** | Select text on any page → click the floating button |

## What Gets Sent

By default, Context Bro extracts the page and compiles a JSON payload like:

```json
{
  "title": "How to fix CORS in Vite",
  "url": "https://stackoverflow.com/q/12345",
  "content": "The full article text in markdown...",
  "author": "johndoe",
  "domain": "stackoverflow.com",
  "wordCount": 1234,
  "clippedAt": "2026-02-22 14:30"
}
```

You can fully customize this shape using [Templates](/guides/templates/).
