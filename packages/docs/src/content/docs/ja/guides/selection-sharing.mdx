## 選択テキストの共有方法

ウェブページでテキストを選択すると、3つの方法で選択テキストを共有できます：

### フローティングボタン

選択テキストの近くに小さなボタンが表示されます。クリックすると、選択テキストがデフォルトエンドポイントに即座に送信されます。

選択を解除するとボタンは自動的に非表示になります。

### キーボードショートカット

`Ctrl+Shift+K`（Macは `Cmd+Shift+K`）で現在の選択を共有します。

このショートカットはブラウザの拡張機能設定でカスタマイズ可能（`chrome://extensions/shortcuts`）。

### コンテキストメニュー

選択テキストを右クリック → **Share selection to Context Bro**。

## 共有される内容

選択テキストを共有すると、`{{selection}}`と`{{selectionHtml}}`テンプレート変数に選択テキストが入ります。`{{content}}`変数には引き続きページ全体のコンテンツが含まれます。

選択テキストのみを送信するテンプレートが必要な場合は、**Selection Only** プリセットを使用：

```json
{
  "selection": "{{selection}}",
  "sourceUrl": "{{url}}",
  "sourceTitle": "{{title}}",
  "clippedAt": "{{date}} {{time}}"
}
```

## ページ全体 vs 選択テキスト

| 変数 | 選択テキストあり | 選択テキストなし |
|------|-----------------|-----------------|
| `{{content}}` | ページ全体のコンテンツ | ページ全体のコンテンツ |
| `{{selection}}` | 選択テキスト（Markdown） | 空文字列 |
| `{{selectionHtml}}` | 選択テキスト（HTML） | 空文字列 |

`{{content}}`と`{{selection}}`は常に利用可能 — どちらを含めるかはテンプレートが決定します。
