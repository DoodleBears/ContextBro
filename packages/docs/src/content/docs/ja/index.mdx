## Context Broは何をするの？

Context Broは、**ウェブページをクリップして任意のAPIエンドポイントに送信する**ブラウザ拡張機能です — AI Agentワークフロー向けに設計されています。

ノートアプリに保存する従来のウェブクリッパーとは異なり、Context Broは**あなた自身のAPI**に構造化データを送信し、AI Agentがブラウジング内容にリアルタイムでアクセスできるようにします。

## 主な機能

- **ワンクリック共有** — ポップアップ、コンテキストメニュー、キーボードショートカット、フローティングボタン
- **7つのプリセットテンプレート** — 汎用、GitHub PR、Stack Overflow、ニュース、Reddit、YouTube、選択テキスト
- **定期自動共有** — 設定可能な間隔でAllowlistドメインを自動抽出（重複排除あり）
- **プラットフォームアダプター** — TwitchとYouTubeライブチャット（優先度サンプリング）
- **完全オープンソース** — MITライセンス、サーバーサイドコンポーネントなし
