## Context Broは何をするの？

Context Broは、**日常のブラウジング行動を構造化コンテキストとして公開する**ブラウザ拡張機能です — シンプルなPOSTリクエストで任意のAPIエンドポイントに配信します。

あなたのブラウジングは豊富なコンテキストの源です。Context BroはAI Agent、自動化パイプライン、またはHTTPリクエストを受信できるあらゆるシステムからアクセスできるようにします。

## 主な機能

- **ワンクリック共有** — ポップアップ、コンテキストメニュー、キーボードショートカット、フローティングボタン
- **7つのプリセットテンプレート** — 汎用、GitHub PR、Stack Overflow、ニュース、Reddit、YouTube、選択テキスト
- **自動キャプチャ** — サイトルールごとにフォーカスモード、バックグラウンドポーリング、動的ページの定期再取得、コンテンツ重複排除を設定
- **プラットフォームアダプター** — TwitchとYouTubeライブチャット（優先度サンプリング）
- **完全オープンソース** — MITライセンス、サーバーサイドコンポーネントなし
