## 概要

Context Broは、ウェブページのコンテキストをAIエージェントと共有するためのブラウザ拡張機能です。プライバシーは設計の中核であり、**Allowlistファースト、デフォルトで何も共有しない**モデルに従います。

## 収集するデータ

**なし。** Context Broは、当社のサーバーでデータを収集、送信、保存することはありません。サーバーはありません。すべての設定（エンドポイント、テンプレート、サイトルール）は `chrome.storage.local` でブラウザにローカル保存されます。

## あなたが共有を選択したデータ

ページを明示的に共有（ポップアップ、コンテキストメニュー、キーボードショートカット、またはフローティングボタン経由）すると、拡張機能は：

- アクティブタブからページコンテンツを抽出（タイトル、URL、メタ説明、著者、[Defuddle](https://github.com/kepano/defuddle)経由の本文、選択テキスト、Schema.org構造化データ、CSSセレクタークエリ）
- 設定されたテンプレートでコンパイル
- コンパイルされたペイロードを**あなた自身のAPIエンドポイント** — あなたが設定したURLに送信

エンドポイントを設定して共有アクションをトリガーしない限り、拡張機能はどこにもデータを送信しません。

## 自動キャプチャ（定期共有）

自動キャプチャを有効にすると、拡張機能は定期的にコンテンツを抽出しますが、**あなたがサイトルールに明示的に追加したドメインのみ**が対象です。コンテンツはローカル保存のSHA-256ハッシュで重複排除され、未変更のページの送信を回避します。ハッシュは設定からいつでもクリアできます。

## ライブストリームアダプター

TwitchおよびYouTubeライブページでは、オプションのアダプターがブラウザに表示されるチャットメッセージを監視します。チャットバッチは設定されたエンドポイントに送信されます。アダプターはマッチするページでのみ有効化され、それらのサイトで拡張機能を読み込まないことで無効にできます。

## 収集されないデータ

- 分析やテレメトリなし
- 閲覧履歴なし
- パスワードやフォームフィールドの内容なし
- シークレットタブからのデータなし
- サーバーサイドストレージなし

## サードパーティサービス

Context Broは、サードパーティの分析、トラッキング、広告サービスを一切統合していません。ネットワークリクエストはあなたが設定したエンドポイントへのもののみです。

## データ保持

すべてのデータはブラウザにローカル保存されます。拡張機能をアンインストールすると、保存されたすべてのデータが削除されます。また、設定からいつでもデータをクリアできます。

## 権限の説明

| 権限 | 必要な理由 |
|------|-----------|
| `activeTab` | 共有クリック時に現在のタブの内容を読み取る |
| `scripting` | コンテンツ抽出スクリプトをページに注入 |
| `storage` | 設定をローカルに保存 |
| `alarms` | 定期共有タイマーを駆動 |
| `tabs` | 定期抽出のために開いているタブを照会 |
| `contextMenus` | 右クリックメニューに「Share to Context Bro」を追加 |
