## インストール

### 方法A：ストア

- **Chrome**：[Chrome ウェブストア](https://chromewebstore.google.com/detail/context-bro/fgiegklhgcecdiephfipnjdhaojlhgff) — **Chromeに追加**をクリック
- **Firefox**：[Firefox アドオン](https://addons.mozilla.org/firefox/addon/context-bro/) — **Firefoxに追加**をクリック

### 方法B：直接ダウンロード（最新版）

最新ビルドを直接ダウンロード — ストア審査の遅延なし：

- [Chrome zip](https://assets.contextbro.app/releases/context-bro-latest-chrome.zip) — 解凍後、`chrome://extensions` で**デベロッパーモード** → **パッケージ化されていない拡張機能を読み込む**
- [Firefox zip](https://assets.contextbro.app/releases/context-bro-latest-firefox.zip) — 解凍後、`about:debugging` → **一時的なアドオンを読み込む**

### 方法C：ソースからビルド

```bash
git clone https://github.com/DoodleBears/ContextBro.git
cd ContextBro
pnpm install
pnpm run build
```

`chrome://extensions` で `.output/chrome-mv3/` ディレクトリを読み込みます。

## 最初のエンドポイントを設定

1. Context Broアイコン → **設定**（歯車アイコン）をクリック
2. **Endpoints** タブに移動
3. **+ Add endpoint** をクリック
4. 名前（例：「マイAPI」）、POST URL、ヘッダーを入力
5. **有効** に切り替え
6. **Save** をクリック

テスト用には `https://httpbin.org/post` を使用できます — 送信内容がすべてエコーバックされます。

## ページを共有

エンドポイントが設定されたら、いくつかの方法で共有できます：

| 方法 | 操作 |
|------|------|
| **ポップアップ** | 拡張機能アイコン → **Share** |
| **コンテキストメニュー** | 右クリック → **Share page to Context Bro** |
| **キーボードショートカット** | `Ctrl+Shift+K`（Macは `Cmd+Shift+K`） |
| **フローティングボタン** | テキストを選択 → フローティングボタンをクリック |

## 送信される内容

デフォルトでは、Context Broはページを抽出して以下のようなJSONにコンパイルします：

```json
{
  "title": "How to fix CORS in Vite",
  "url": "https://stackoverflow.com/q/12345",
  "content": "記事全文のMarkdownテキスト...",
  "author": "johndoe",
  "domain": "stackoverflow.com",
  "wordCount": 1234,
  "clippedAt": "2026-02-22 14:30"
}
```

[テンプレート](/ja/guides/templates/)でこの形式を完全にカスタマイズできます。
