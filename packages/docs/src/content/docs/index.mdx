## What does Context Bro do?

Context Bro is a browser extension that **exposes your everyday browsing behavior as structured context** — delivered to any API endpoint via a simple POST.

Your browsing is a rich source of context. Context Bro makes it accessible to AI agents, automation pipelines, or any system that can receive HTTP requests.

## Key Features

- **One-click share** — popup, context menu, keyboard shortcut, or floating button
- **7 preset templates** — General, GitHub PR, Stack Overflow, News, Reddit, YouTube, Selection
- **Auto-capture** — per-site rules with focused mode, background polling, refetch for dynamic pages, and content dedup
- **Platform adapters** — Twitch and YouTube live chat with priority sampling
- **Fully open source** — MIT License, no server-side components
