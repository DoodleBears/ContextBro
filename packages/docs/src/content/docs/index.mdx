## What does Context Bro do?

Context Bro is a browser extension that **clips web pages and sends browsing context to any API endpoint** — designed for AI agent workflows.

Unlike traditional web clippers that save to a note app, Context Bro sends structured data to **your own API**, giving AI agents real-time access to what you're browsing.

## Key Features

- **One-click share** — popup, context menu, keyboard shortcut, or floating button
- **7 preset templates** — General, GitHub PR, Stack Overflow, News, Reddit, YouTube, Selection
- **Scheduled auto-sharing** — allowlisted domains at configurable intervals with content dedup
- **Platform adapters** — Twitch and YouTube live chat with priority sampling
- **Fully open source** — MIT License, no server-side components
