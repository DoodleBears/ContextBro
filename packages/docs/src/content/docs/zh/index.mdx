## Context Bro 做什么？

Context Bro 是一个浏览器扩展，**将网页内容提取并发送到任意 API 端点** —— 专为 AI Agent 工作流设计。

与传统的网页剪藏工具（保存到笔记应用）不同，Context Bro 将结构化数据发送到**你自己的 API**，让 AI Agent 实时获取你正在浏览的内容。

## 核心功能

- **一键分享** —— 弹出窗口、右键菜单、键盘快捷键或浮动按钮
- **7 个预设模板** —— 通用、GitHub PR、Stack Overflow、新闻、Reddit、YouTube、选区
- **定时自动分享** —— 在可配置的间隔内自动提取允许列表域名的内容（含去重）
- **平台适配器** —— Twitch 和 YouTube 直播聊天，优先级采样
- **完全开源** —— MIT 许可证，无服务端组件
