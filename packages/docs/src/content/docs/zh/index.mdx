## Context Bro 做什么？

Context Bro 是一个浏览器扩展，**将你日常的浏览行为作为结构化上下文暴露出来** —— 通过简单的 POST 请求发送到任意 API 端点。

你的浏览行为是丰富的上下文来源。Context Bro 让 AI Agent、自动化流水线或任何能接收 HTTP 请求的系统都能访问这些上下文。

## 核心功能

- **一键分享** —— 弹出窗口、右键菜单、键盘快捷键或浮动按钮
- **7 个预设模板** —— 通用、GitHub PR、Stack Overflow、新闻、Reddit、YouTube、选区
- **自动捕获** —— 按站点规则配置聚焦模式、后台轮询、动态页面定时刷新和内容去重
- **平台适配器** —— Twitch 和 YouTube 直播聊天，优先级采样
- **完全开源** —— MIT 许可证，无服务端组件
