## 安装

### 方式 A：应用商店

- **Chrome**：[Chrome 应用商店](https://chromewebstore.google.com/detail/context-bro) — 点击 **添加到 Chrome**
- **Firefox**：[Firefox 附加组件](https://addons.mozilla.org/firefox/addon/context-bro/) — 点击 **添加到 Firefox**

### 方式 B：直接下载（最新版）

直接下载最新构建 — 始终保持最新，无需等待商店审核：

- [Chrome zip](https://assets.contextbro.app/releases/context-bro-latest-chrome.zip) — 解压后在 `chrome://extensions` 开启 **开发者模式** → **加载已解压的扩展程序**
- [Firefox zip](https://assets.contextbro.app/releases/context-bro-latest-firefox.zip) — 解压后在 `about:debugging` → **加载临时附加组件**

### 方式 C：从源码构建

```bash
git clone https://github.com/DoodleBears/ContextBro.git
cd ContextBro
pnpm install
pnpm run build
```

然后在 `chrome://extensions` 中以开发者模式加载 `.output/chrome-mv3/` 目录。

## 配置第一个端点

1. 点击 Context Bro 图标 → **设置**（齿轮图标）
2. 进入 **Endpoints** 标签页
3. 点击 **+ Add endpoint**
4. 输入名称（如"我的 API"）、POST URL 和所需的 Headers
5. 切换为 **已启用**
6. 点击 **Save**

测试时可以使用 `https://httpbin.org/post` —— 它会回显你发送的所有内容。

## 分享页面

配置好端点后，有多种分享方式：

| 方式 | 操作 |
|------|------|
| **弹出窗口** | 点击扩展图标 → **Share** |
| **右键菜单** | 右键 → **Share page to Context Bro** |
| **快捷键** | `Ctrl+Shift+K`（Mac 上为 `Cmd+Shift+K`） |
| **浮动按钮** | 选中任意页面文本 → 点击浮动按钮 |

## 发送的内容

默认情况下，Context Bro 提取页面并编译为如下 JSON：

```json
{
  "title": "How to fix CORS in Vite",
  "url": "https://stackoverflow.com/q/12345",
  "content": "完整文章的 Markdown 文本...",
  "author": "johndoe",
  "domain": "stackoverflow.com",
  "wordCount": 1234,
  "clippedAt": "2026-02-22 14:30"
}
```

你可以通过[模板配置](/zh/guides/templates/)完全自定义这个格式。
